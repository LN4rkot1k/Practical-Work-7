1) Что такое рекурсивная подпрограмма?

Рекурсия — это такой способ организации вспомогательного алгоритма (подпрограммы), при котором эта подпрограмма в ходе выполнения ее операторов обращается сама к себе.

2) Что такое итерация?

Итерация— это организация обработки данных, при которой действия повторяются многократно, не приводя при этом к вызовам самих себя.

3)Что такое стек? 

Стек — это структура данных, в которой элементы хранятся в порядке поступления.

4)Составьте рекурсивный и итерационный алгоритмы нахождения N!.

Рекурсивный алгоритм. 

def fact(n):
    if n == 0:
        return 1
    return fact(n - 1) * n


print(fact(5))

Итерационный алгоритм. 

def fact1(n):
    result = 1
    while n > 1:
        result = result * n
        n -= 1
    return result


print(fact1(5))

5)Объясните термин "косвенная рекурсия"?

Косвенная рекурсия – это когда функция X вызывает функцию Y, которая в свою очередь вызывает функцию X.
